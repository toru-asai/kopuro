# SetUp
## install rbenv
- 省略

#### install bundler ####

```
gem install bundler -v 2.0.1
gem list bundler
bundle -v
```

#### Podのインストール ####

```
bundle _2.0.1_ install --path vendor/bundle
```



## install cocoapods
- $ bundle install --path vendor/bundle
- $ bundle exec pod update
- $ bundle exec pod install
